# 💰 Stable-Fin

## Overview

Stable-Fin is a retirement planning and financial sustainability calculator that helps users estimate whether their current savings and investment strategy will support them after retirement.

The application considers:

- Current salary
- Monthly expenditure
- Existing savings/investments
- Monthly investments
- Incremental increases in investments over time
- Retirement age (default: 60 years)
- Inflation impact
- Purchasing Power Parity (PPP) adjustment

## Features

✅ Calculates projected retirement corpus

✅ Supports periodic investment increases based on expected salary growth

✅ Adjusts future value using inflation assumptions

✅ Estimates retirement expenses at retirement age

✅ Determines how long retirement savings will last

✅ Displays sustainability duration in years/months

✅ Warns users when retirement funds may be insufficient

## Requirements

- Java 17

## Usage

Input can be provided in either of the following ways:

1. Through console input
2. By updating values directly within the source code
